# PharmAllele-UPF
This repository contains the PharmAllele project, a web application developed as part of the course of Database and Web Development (MSc Bioinformatics for Health Sciences, UPF/UB).

This web app allows to make pharmacogenomic consults to a database where the relation between gene variants and drugs is stored. The original data come from the ClinPGx database.

Currently the server where the web app should be isn't publicly accesible, as it belongs to the UB.
